## Body Parameters

<ParamField body="size" type="number">
  Maximum size of the JSON object that will be returned (maximum 10 000) 
</ParamField>

<ParamField body="from" type="number">
  The `next` value from the last response.
</ParamField>

<ParamField body="query" type="object">
  One of the supported queries.
  <CodeGroup>
  ```json Domain Query
  {
    "type": "domain",
    "fqdn": "<string>"
  }
  ```

  ```json Password Query
  {
    "type": "secret",
    "secret": "<string>"
  }
  ```

  ```json Keyword Query
  {
    "type": "keyword",
    "keyword": "<string>"
  }
  ```
  </CodeGroup>
</ParamField>
