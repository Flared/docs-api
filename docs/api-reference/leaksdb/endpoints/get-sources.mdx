## Query Parameters

<ParamField query="size" type="number">
  Maximum size of the JSON object that will be returned (maximum 10 000).
</ParamField>

<ParamField query="from" type="string">
  Cursor from which you want to continue paginating.
  This value comes from the `next` field in the response.
  For example `WzFd` the example response.
</ParamField>
