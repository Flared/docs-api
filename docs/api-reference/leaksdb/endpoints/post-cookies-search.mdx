## Body Parameters

<ParamField body="domain" type="string">
  The exact domain for which you want to search cookies. Subdomains will not be included.
</ParamField>

<ParamField body="size" type="number">
  The number of results to fetch. (max: 10)
</ParamField>

<ParamField body="from" type="string">
  The `next` value from the last response.
</ParamField>

<ParamField body="paths" type="string[]">
  A list of paths to filter for specific cookies, for example `['/', '/login']`. (max: 50)
</ParamField>

<ParamField body="expires_after" type="string">
  The ISO-formatted datetime string to filter for cookie expiration, for example `2024-10-29T17:50:44.237148+00:00`.
</ParamField>
