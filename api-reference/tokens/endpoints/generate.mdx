---
title: 'Generate an API Token'
api: 'POST https://api.flare.io/tokens/generate'
authMethod: 'basic'
---


Tokens are used to interact with the Flare API.
They can be generated using an API key provided via HTTP basic auth.

API tokens expire after 1 hour.

For more information about authentication, see the [API Authentication Guide](/introduction/authentication/).

<ResponseExample>

```json Response Schema
{
  "refresh_token_exp": <number (unix timestamp)>,
  "token": "api-token"
}
```

```json Response Example
{
  "refresh_token_exp": 1723673997,
  "token": "i-am-an-example-api-token"
}
```

</ResponseExample>

## Basic Auth

The `/generate` endpoint uses HTTP basic auth with the `Authorization` header.
- The user must be empty.
- The password must be your API key.

## Parameters

<ParamField body="tenant_id" type="number">
  The tenant you want to authenticate to.
  
  When unspecified, the API token will be generated for your default tenant.
</ParamField>

## Response Fields

<ResponseField name="token" type="string">
  The generated API token.
</ResponseField>

<ResponseField name="refresh_token_exp" type="number">
  Unix timestamp of the expiration of the refresh token.
  
  (The refresh token is returned in cookies).
</ResponseField>
